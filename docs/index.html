LibriaForge Mail Cleanup is a free, open-source desktop application that connects to your Gmail or Outlook account and helps you bulk-delete, archive, and organise emails using AI-powered classification.

How it works

⚡

Keyword pre-pass

Instantly classifies obvious noise — marketing, noreply, bulk senders — with no API calls needed.

🤖

AI classification

Claude Haiku analyses remaining senders by name and subject lines, suggesting delete, archive, or keep.

📁

Category folders

Archived emails are automatically routed to named folders: Newsletters, Receipts, Alerts, Dev, and more.

💾

Saved rules

Save decisions per sender. Next run applies them instantly — no repeated prompts.

🔕

Unsubscribe + Delete

Follows the List-Unsubscribe header automatically, then deletes the emails.

🔒

Private by design

No server. Tokens stored locally. Only sender names and subject lines ever leave your machine.

Supported providers

Authenticate with your own account — no passwords are ever stored.

✉ Gmail — OAuth 2.0
✉ Outlook — Device code flow
